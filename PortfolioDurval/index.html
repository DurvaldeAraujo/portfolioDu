Trazendo os Boletins Informativos da FATEC Lins: Explorando Conceitos de Escrita para Web.

O trabalho proposto envolve a criação de uma página para exibir os boletins informativos da FATEC Lins, conforme mostrado na imagem 1, durante a disciplina de criação de conteúdo para web. A página criada deve abordar os conceitos discutidos no semestre de 2019, como escrita para web, uso de etiquetas de comunicação, planejamento do projeto de escrita, usabilidade focada no conteúdo, utilização da pirâmide invertida, estratificação de dados, redação e otimização para mecanismos de busca (SEO). Durante o desenvolvimento, todo o boletim informativo do site da FATEC Lins foi analisado, filtrando e selecionando as informações relevantes de acordo com as técnicas textuais ensinadas em sala de aula, para melhor adequação a cada projeto.

Explorando a Identidade Visual de uma Pizzaria: O Trabalho de Design no Curso de Sistemas para Internet

×

Neste capítulo, relata-se sobre um trabalho realizado no segundo semestre do curso de Sistemas para Internet, na disciplina de Prática de Design. O objetivo era criar uma empresa fictícia e desenvolver sua identidade visual, incluindo camiseta, logotipo, cartão fidelidade, panfleto e cardápio, tendo em mente que a empresa seria uma pizzaria. Para a criação, foi utilizado o software CorelDraw, amplamente utilizado na montagem de diversos tipos de designs, devido aos seus recursos e capacidade de trabalhar com vetores, garantindo eficiência e preservação da qualidade das imagens escolhidas durante o redimensionamento..

Acessibilidade na Internet: Garantindo a Igualdade de Acesso e Oportunidades para Todos

×

Acessibilidade na Internet refere-se à garantia de que todas as pessoas, incluindo aquelas com deficiência, possam utilizar e desfrutar plenamente da web. É um conceito que envolve a criação de sites e aplicativos acessíveis, que possam ser percebidos, compreendidos, navegados e interagidos por todas as pessoas, independentemente de suas habilidades ou limitações.Os criadores de conteúdo que desejam tornar seus sites mais acessíveis podem utilizar recursos de programação e ferramentas específicas para avaliar a acessibilidade. Isso permite identificar áreas que precisam ser melhoradas e garantir que o site atenda aos padrões de acessibilidade estabelecidos pela World Wide Web Consortium (W3C) e outras organizações relevantes.

Explorando o User Experience (UX) e criando um protótipo de portfolio interativo

×

No quarto semestre do curso, na disciplina de Projeto de Navegação e Interação, é abordado o conceito de User Experience (UX) e sua importância na atualidade. UX significa Experiência do Usuário e engloba a experiência que uma pessoa tem ao utilizar objetos ou sistemas, como websites, aplicativos de telefone, TVs e até mesmo objetos cotidianos, como uma caneta.Com base nos princípios do UX, foi proposta a criação de uma interface interativa com o usuário, especificamente um protótipo de portfolio para ser apresentado como trabalho de conclusão de curso (TCC). Para a criação desse protótipo, foi escolhida a ferramenta de design Figma, que possui diversas funcionalidades para a montagem do projeto de alta fidelidade, contendo todas as páginas web necessárias. Além disso, foi necessário desenvolver um processo escrito abordando temas como compreensão, planejamento, requisitos funcionais e não funcionais, design conceitual e design físico do portfolio.

Explorando o Desenvolvimento de Aplicativos Móveis com Apache Cordova: Uma Jornada Multiplataforma

×

Neste capítulo do curso "Tópicos especiais em sistemas para Internet II", exploramos o desenvolvimento de aplicativos móveis utilizando o framework Apache Cordova. Através do uso de tecnologias web padrão, como HTML5, CSS3 e Javascript, aprendemos a estrutura básica para a criação de um aplicativo multiplataforma. O Apache Cordova oferece uma variedade de plugins que podem ser adicionados ao aplicativo, como Status Battery, Camera, Device, Dialogs, File e Geolocation. Através do arquivo Config.xml, é possível personalizar a configuração do aplicativo, incluindo ícones, nome e endereços. O LocalStorage é utilizado como uma opção de armazenamento de dados no navegador.

Explorando o Git: Controle de Versão e Colaboração Eficiente no Desenvolvimento de Software

×

No sexto semestre do curso de Tópicos especiais em sistemas para Internet III, foi abordado o tema do Git e sua importância no desenvolvimento de software. O Git é um sistema de controle de versão amplamente utilizado, que permite rastrear alterações, colaborar com outros desenvolvedores e manter um histórico detalhado do código.Durante o curso, foram explorados os conceitos básicos do Git, desde a instalação até a criação de repositórios, adição de arquivos, criação de ramificações e fusões. Foram apresentados comandos essenciais, como git init, git clone, git add, git commit, git push, git pull, git branch, git checkout e git merge. Também foram discutidas práticas recomendadas, como o uso adequado de mensagens de commit, organização de ramificações e resolução de conflitos.
